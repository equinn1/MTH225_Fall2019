Mean-only

 R code R code normal.stan This is the model that more complicated models reduce to when none of the independent variables are good predictors of the dependent variable. In effect, it says the best estimate of the mean of the dependent variable is just the mean of the sample.
Simple regression

 R code R code Stan model file Assumes a linear relationship between the dependent and independent variables, with superimposed noise so everything doesn't fall on a straight line. If there is a strong linear relationship, the unexplained variation in the dependent variable is reduced because the independent variable provides information about it.
Multiple regression

 R code R code Stan model file Assumes a linear relationship between the dependent and independent variables, with superimposed noise so everything doesn't fall on a straight line. If there is a strong linear relationship, the unexplained variation in the dependent variable is reduced because the independent variables provide information about it. Some predictors can be binary or "dummy" variables.
One-way ANOVA 3 levels

 R code R code Stan model file The fitted values in a one-way or single-factor ANOVA are just the sample means at each level of the factor.
Two-way ANOVA no interaction

 R code R code Stan model file This model has two factors or classification variables. The distinguishing feature is that the difference between levels of the first factor is the same for all levels of the second factor (and vice versa).
Two-way ANOVA with interaction

 R code R code Stan model file While the standard parameterization is overly complicated, the fitted values for this model are just the means of each cell (each unique combination of levels for the two factors). The distinguishing characteristic is that the difference between levels of the first factor changes depending on the level of the second factor (and vice versa).
Covariance - common slope

 R code R code Stan model file The independent variables for this model are one factor and one continuous predictor. The fitted values are a series of parallel regression lines, one for each level of the factor. The graphic illustrates the fitted values when the factor has two levels. The lines are parallel because there is only one slope parameter that is common to all levels of the factor.
Covariance - unequal slopes

 R code R code Stan model file The independent variables for this model are one factor and one continuous predictor. The fitted values are a series of regression lines, one for each level of the factor. The graphic illustrates the fitted values when the factor has two levels. This model allows different slope parameters at each level of the factor, so if this model provides a better fit than the model with a common slope for all levels, the lines will not be parallel.
